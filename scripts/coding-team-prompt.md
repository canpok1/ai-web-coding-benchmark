## 実装条件
- 出力先: work/generated/ ディレクトリ内にすべてのファイルを配置すること。
- 技術スタック: HTML, CSS, JavaScript を使用し、モダンなコーディングを行ってください。
- 構成ファイル: index.html を基準に、必要に応じて CSS や JavaScript ファイルを作成してください。
- 分割のルール:
  - 1ファイルあたりの行数は100行程度を目安にしてください。
  - 積極的にファイルを分割し、モジュール化を図ってください。
  - モジュールはWebコンポーネントとして実装してください。
  - CSSはモジュール内にカプセル化してください。

## 禁止事項
- work/generated/ 以外のディレクトリにファイルを生成・変更しないでください。
- 外部の複雑なライブラリ（React, Vue等）は使用せず、Vanilla JSで実装してください。

## 実装のコツ
- 全体から細部へと段階的に実装を進めることをおすすめします。
- 見本のHTMLがある場合、次の流れで実装を進めることをおすすめします。
  1. 見本ページのHTML/CSSと同じ構造を作成する。
  2. ルールに従ってファイルを分割する。
- 見本のHTMLがない場合は、次の流れで実装を進めることをおすすめします。
  1. 背景を実装する。
  2. レイアウトを実装する。
  3. 細部のスタイルを実装する。
